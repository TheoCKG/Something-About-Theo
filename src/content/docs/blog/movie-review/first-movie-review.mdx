> 这是一个示例影评模版，你可以直接改片名与内容，写成自己的第一篇影评。

## 一、影片信息

- 片名：
- 导演 / 编剧：
- 主演：
- 类型与时长：

## 二、无剧透印象（可选）

如果你希望这篇文章兼具“安利”作用，可以先用一小段**无剧透**的印象来告诉读者：

- 这部片子适合什么样的观众
- 大概是什么气质（轻松 / 压抑 / 温情 / 猎奇 / 黑色幽默等）

## 三、故事与主题（含轻度剧透）

在不过多剧透关键反转的前提下，简单梳理：

- 故事发生在怎样的背景下？
- 主角在追求什么？面临的核心冲突是什么？
- 你认为影片真正想讨论的主题是什么？（家庭、身份认同、阶级、选择等）

## 四、人物与表演

- 哪个角色最打动你？为什么？
- 有没有因为表演而“加分”或“出戏”的地方？

## 五、视听语言与细节

可以从以下角度挑一两个展开：

- 镜头调度、剪辑节奏、配乐与音效的运用
- 某些反复出现的意象 / 色彩 / 构图
- 你注意到的彩蛋或隐喻

## 六、个人收获与共鸣

最后可以写写：

- 这部作品和你的哪些个人经验产生了共鸣
- 看完之后，有没有哪句话或哪一幕一直留在脑海里
